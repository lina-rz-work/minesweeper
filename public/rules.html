Game Rules


1. Goal of the game: the goal of the game is to uncover all cells that do not contain mines.

2. Game Modes: Choose the difficulty level in the "Game Mode" section. You can also enable hints in the "Game Hints" section.

3. Game Field: The game field is represented by a square grid of cells. Above the game field, there is an indicator for flags, a game timer, and a button with an eye that provides hints about mines.

4. Gameplay:
- Opening a cell: Left-clicking on a cell can open it. If there is no mine under the opened cell, it reveals a number (indicating the number of mined neighbors) or an "unmined" area.
- Flagging a cell: Right-clicking on a cell allows you to place a flag where you suspect a mine.

5. Game Completion:
- The game ends in defeat if a cell with a mine is opened.
- The game ends successfully if all cells without mines are opened.

6. New Game:
- Clicking the "New Game" button closes the game field and returns to the main page with the menu.
